Task 4 completed: Sentiment Analysis on textual data using NLP techniques
This project performs Sentiment Analysis on Twitter data using Natural Language Processing and Machine Learning to classify tweets as Positive or Negative. The model was trained using a Naive Bayes algorithm and achieved around 95% accuracy, demonstrating how NLP can be used to analyze customer opinions and social media feedback.

🛠 Technologies Used

Python

Pandas & NumPy – Data handling

NLTK – Text preprocessing

Scikit-learn – Machine learning model

Matplotlib & Seaborn – Visualization

⚙️ Project Workflow

Data Loading: Imported Twitter sentiment dataset.

Text Preprocessing: Cleaned text by removing punctuation, stopwords, and applying stemming.

Feature Extraction: Converted text into numerical features using CountVectorizer.

Model Training: Trained a Naive Bayes classifier.

Evaluation: Measured accuracy and generated confusion matrix.

Prediction: Classified new tweets as Positive or Negative.
